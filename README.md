# Email Co-Composer

An AI-powered email writing assistant that helps users communicate more effectively by specifying their communicative intent before generating text.

## Getting Started

### Prerequisites

- Node.js 18 or higher
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd hcai-email-co-composer
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```
